<h1 align="center">FileMaster</h1>
<h4 align="center">BenjiaH</h4>

## Requirements
- comtypes

## Features
- 将当前目录文件按文件类型分类
- 将当前目录文件按文件名分类
- 将当前目录文件夹解散
- Word文档批量转换为PDF文件
- Power Point文档批量转换为PDF文件
- 计算文件MD5,SHA1信息
